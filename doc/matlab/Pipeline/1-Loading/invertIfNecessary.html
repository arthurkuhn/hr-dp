invertIfNecessary

PURPOSE

invertIfNecessary Decides whether or not to invert the ECG signal

SYNOPSIS

function [ res ] = invertIfNecessary( rawEcg, fs, windowInSeconds )

DESCRIPTION 

invertIfNecessary Decides whether or not to invert the ECG signal
   Based on the analysis of a 10mns signal segment, we decide whether or
   not to invert the original signal.

CROSS-REFERENCE INFORMATION

This function calls:
  • preprocessingNew every second of the ECG signal was normalized by the standard deviation of the signal in that second.
  • kota KOTA QRS detection using Hilbert Transform
